# XAU Tray

XAU Tray 是一款轻量级菜单栏/系统托盘应用，实时监控 AllTick 行情并在状态栏展示最新价格，悬浮提示显示所有品类的价格与趋势。

XAU Tray is a lightweight menu bar/system tray app that monitors AllTick quotes in real time and shows the latest price in the status area, with a tooltip for all tracked symbols.

## 功能 Features

- 状态栏实时价格与涨跌趋势 / Live price and trend in the status area
- 轮播或固定展示指定品类 / Rotate or pin a fixed symbol
- 自定义品类列表与快捷预设 / Custom symbol list with quick presets
- 可选使用系统代理（启用且可用时） / Optional system proxy usage (when enabled and available)
- 本地保存 Token 设置 / Local token storage

## 适用平台 Platforms

- macOS（菜单栏）/ macOS (menu bar)
- Windows（系统托盘）/ Windows (system tray)
- Linux（系统托盘）/ Linux (system tray)

## 依赖 Requirements

- Node.js 18+ 与 pnpm / Node.js 18+ and pnpm
- Rust stable 工具链 / Rust toolchain (stable)

## 开发 Development

```bash
pnpm install
pnpm tauri dev
```

## 构建 Build

```bash
pnpm install
pnpm tauri build
```

构建产物由 Tauri 输出到 `src-tauri/target/release/`（与平台有关）。

Build artifacts are generated by Tauri in `src-tauri/target/release/` (platform-dependent).

## 配置 Configuration

- AllTick Token：用于获取行情 / Required for fetching quotes
- 刷新间隔：免费模式最低 10 秒 / Minimum 10 seconds for free plan
- 展示方式：轮播或固定 / Display mode: rotate or fixed
- 系统代理：启用后可使用系统代理 / System proxy: enable to use OS proxy settings when available
